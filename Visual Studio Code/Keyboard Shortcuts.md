# 1. Navigation

- **Move Cursor:**
  - Arrow keys

- **Move Word-wise:**
  - Ctrl + arrow keys

- **Select Text:**
  - Shift + arrow keys

- **Select Word-wise:**
  - Ctrl + Shift + arrow keys

- **Page Navigation:**
  - Ctrl + Page Up/Page Down

- **Go to Line:**
  - Ctrl + G, then enter line number

- **Go to Beginning/End of File:**
  - Ctrl + Home/End

- **Toggle Sidebar:**
  - Ctrl + B

- **Toggle Terminal:**
  - Ctrl + `

- **Split Editor:**
  - Ctrl + \

- **Quick Open File:**
  - Ctrl + P, then start typing filename

- **Search in File:**
  - Ctrl + F

- **Search and Replace:**
  - Ctrl + H

- **Undo/Redo:**
  - Ctrl + Z, Ctrl + Y

- **Copy/Cut/Paste:**
  - Ctrl + C, Ctrl + X, Ctrl + V

- **Save File:**
  - Ctrl + S

- **Save As:**
  - Ctrl + Shift + S
  - Ctrl + Shift + S

- **Close File:**
  - Ctrl + W
